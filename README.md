# Mawu CMS

Mawu is a CMS for Arcturus MorningStar, with updated HoloCMS(2008) system.

Mawu was made by MiguelGonzalezAravena, MarcoCuel, Wulles, Skorp and WagnerDiniz.

## Compatible

*  Arcturus
*  PHP 7
*  MariaDB 10.4.14
